## PubIP | Get Your Public IP Address

![PubIP](Screenshot.png?raw=true "PubIP")


## Clone and Use !
Use this command on Linux Terminal !
```
git clone https://github.com/The404Hacking/PubIP.git
cd PubIP
python PubIP.py
```
